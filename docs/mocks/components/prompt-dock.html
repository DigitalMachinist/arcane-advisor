Prompt Dock

The torn-parchment prompt input. Always two lines tall. In the live UI it's anchored to the viewport bottom; here it's rendered inline at canonical width so the visual details are easier to review. States below isolate empty / drafting / sealed / narrow.

The default appearance at the start of a session or after a round has resolved. Placeholder text rotates between period-appropriate suggestions.
