Components · Sourcebook Selector

Sourcebook Selector

A header-level control for enabling or disabling official D&D 5e wizard sourcebooks (PHB, XGE, TCE, FTD, SCC). Collapsed: a stacked-tomes icon with a live count. Expanded: a parchment dropdown with one row per book. Minimum one book must remain enabled; the last toggle refuses. Canonical Spec 8 behaviour.

Collapsed · All five enabled
Default state

The baseline. On first visit and for most returning users. Icon plus the live count of enabled books. Quiet register; sits among the other header affordances (rounds pips, new-consultation button).

Collapsed · 5 of 5
Collapsed · Subset enabled
Partial selection

When the user has disabled one or more books the count shows "N of 5". This is the honest signal — the icon does not change state or colour, only the label does.

Collapsed · 3 of 5
Collapsed · Single book
Minimum selection

When only one book is active the label reads "1 book" (singular). This is the lower-bound state — the dropdown will refuse to let the user disable the last one.

Collapsed · 1 of 5
Collapsed · Hover & open
Affordance states

On hover, the border and label lift to the arcane amber. The open state (right) additionally gains a faint warm fill and the icon picks up a small glow — signalling the panel is live beneath it.

Left: hover · Right: open (panel anchor)
Expanded · All five enabled
Dropdown · default

The dropdown is a torn-edged parchment strip anchored to the affordance. Five rows in canonical order (PHB → SCC), each with the code, full title, and a toggle. Clicking anywhere on the row toggles the book; the footnote reminds the user about the minimum.

Expanded · Refusing the last toggle
Minimum guard

When only one book remains enabled and the user clicks its toggle, the toggle refuses — a small horizontal shake — and the footnote briefly emphasises in warning tone. No modal, no error; the control simply declines the operation.
