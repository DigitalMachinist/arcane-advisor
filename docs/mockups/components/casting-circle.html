Casting Circle & Needle

The atmospheric background: an ornate arcane circle with a compass needle that rotates to point at the active whimsy rune. Rendered at 300px here — in the live UI it scales to 950px, centered and ghosted behind the transcript. Stroke and glow colours cascade from the active whimsy; the needle rotation is set per symbol (tactical +30°, balanced +15°, creative 0°, theatrical −15°, chaotic −30°) so the tip points down toward the matching symbol in the linear dial (CSS rotate is clockwise, so +deg leans left).
